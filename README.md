# ClearAudit 🚀

ClearAudit is an intelligent, AI-powered expense auditing platform built for modern companies. It streamlines the entire expense reporting process by leveraging Google's Gemini AI to automatically extract, categorize, and analyze receipts for corporate policy violations and fraud risks.

This repository is split into two main sections:
- 🎨 **[Frontend](./frontend/README.md)**: The React-based user interface containing the Worker and Chief dashboards.
- ⚙️ **[Backend](./backend/README.md)**: The Node.js API that powers the Gemini AI analysis, Supabase integrations, and core business logic.

## 🛠️ Global Technology Stack
- **Frontend**: React, Tailwind CSS, Vite, Lucide Icons
- **Backend**: Node.js, Express.js, Multer (File Uploads)
- **Database & Auth**: Supabase (PostgreSQL, Authentication, Storage)
- **AI Intelligence**: Google Gemini API (`@google/genai`)

## ⚙️ Quick Start

1. **Clone the repository**
2. Setup the **[Backend](./backend/README.md)** first.
3. Setup the **[Frontend](./frontend/README.md)** second.
4. Enjoy your AI-powered auditing platform!
